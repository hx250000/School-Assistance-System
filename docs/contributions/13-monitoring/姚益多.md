## 遇到的问题和解决

- **问题**：Swagger 文档中未显示 `/health` 接口
  - **解决**：接口代码与路径配置正确，功能可正常访问，Swagger 展示问题不影响接口使用，已完成接口验证。

- **问题**：需要输出符合要求的结构化 JSON 日志
  - **解决**：引入 `logstash-logback-encoder` 并配置 `logback-spring.xml`，实现日志 JSON 格式化输出。

- **问题**：Actuator 监控端点无法访问
  - **解决**：在 `application.yml` 中配置暴露 `health`、`metrics` 等端点，成功获取请求计数、响应时间、错误率指标。
