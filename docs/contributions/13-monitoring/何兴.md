## 遇到的问题和解决

### 问题1

Spring Boot 默认未暴露所有 Actuator 端点。

**解决：**

在application.yml中添加：

```yaml
management:
  endpoints:
    web:
      exposure:
        include: health,info,metrics
  endpoint:
    health:
      show-details: always
```

成功访问监控接口。

### 问题2

初始阶段监控指标数据较少。

**解决：**

通过访问业务接口生成请求数据后，再查看监控指标，成功获取请求计数和响应时间信息。
