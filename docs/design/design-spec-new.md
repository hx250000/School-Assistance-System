# 设计说明
 ## 核心页面
0. 底部导航栏 - 五个主要页面的快速切换
1. 首页 - 任务浏览、搜索功能、任务分类筛选（全部/游戏/生活/学习）
2. 任务详情页 - 完整的任务信息展示、抢任务功能
3. 发布任务页 - 表单式发布，包含标题、描述、类型、人数、积分、截止时间
4. 我的任务 - 三个分类（已发布/已参与/已完成）+ 任务统计卡片
5. 积分商城 - 商品展示、积分余额、兑换功能
6. 用户中心 - 个人信息、积分和信用评分、快捷入口、设置菜单
7. 我的成就 - 个人成就显示界面
## 设计亮点
- 采用渐变色卡片设计，视觉现代化
- 移动端优先的响应式设计（最大宽度适配）
- 任务类型彩色标签（游戏-紫色、生活-绿色、学习-蓝色）
- 积分系统完整展示
- 消息提醒徽章
- 流畅的页面交互和过渡效果
## 设计细节
### 配色方案
整体避免使用过于鲜艳的颜色，使用更柔和的渐变，提升视觉舒适度
- 主色：首页/任务/我的页面顶部采用紫色渐变：
linear-gradient(135deg, #7B61FF 0%, #B85CFF 55%, #FF5AA5 100%)，
发布/商城/成就页面顶部采用橙色渐变：linear-gradient(135deg, #FFB000 0%, #FF7A3D 55%, #FF5A8B 100%)
- 背景：#FAF4AE 

### 字体
### 阴影显示
- 所有边框为精致的阴影
- Hover状态使用更深的阴影
- 按钮使用彩色阴影
### 边框优化
- 保留必要的细边框
- 使用 ring-4 替代厚边框，更加精致
### 圆角显示
- 统一使用 rounded-2xl（大部分卡片）
- 小元素使用 rounded-xl
- 保持一致性
### 背景装饰
- 降低装饰性背景的透明度（opacity-[0.06]）
- 减少过于花哨的装饰元素
- 让界面更加简洁
## 交互说明
1.通过点击底部导航栏切换不同页面
2.点击首页“全部”“游戏”“生活”“学习”不同标签切换不同种类任务
3.在“我的”页面中点击“查看我的成就”或快速导航/我的服务中的“我的成就”，跳转到成就界面
